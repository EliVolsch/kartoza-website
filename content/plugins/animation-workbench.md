## Overview

QGIS Animation Workbench is an in-house Kartoza plugin developed by Tim Sutton, Nyall Dawson, and Jeremy Prior. While QGIS includes the Temporal Manager for time-based data animations, Animation Workbench goes further by enabling camera movements, zoom transitions, and dynamic feature styling.

Create professional-quality animated maps for presentations, social media, or documentary content.

## Key Features

- **Camera Paths**: Define smooth camera movements across your map
- **Zoom Transitions**: Create dramatic zoom in/out effects
- **Feature Animation**: Make features wiggle, pulse, or transform
- **Keyframe Editor**: Precise control over animation timing
- **Export Options**: Render to video or image sequences
- **QGIS Integration**: Works with any QGIS project and styling

## Animation Types

- Fly-through animations
- Zoom reveals
- Pan transitions
- Feature highlighting
- Temporal data playback
- Custom expression-driven animations

## Use Cases

- Conference presentations
- Social media content
- Documentary filmmaking
- Educational materials
- Data storytelling
- Marketing visualisations

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "Animation Workbench"
3. Click **Install Plugin**
4. Access the workbench from the QGIS toolbar
