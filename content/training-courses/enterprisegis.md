## Course Overview

NEEDS UPDATED DESCRIPTION! This course will give you the basic skills and knowledge to get going with establishing an enterprise GIS or transforming or improving your current system. We will use FOSS (Free and Open Source) tools because they are world-class! Experience how a full, professional Enterprise GIS can be achieved with FOSS.

## What You Will Learn

<div class="two-column-list">

- Create and populate a spatial database
- Install a web map server
- Publish OGC map services

<!-- Column break -->

- Compose a web map; view and search it in a desktop and a web client
- Understand some architectural guidelines for a corporate environment
- Administer and troubleshoot the system

</div>
