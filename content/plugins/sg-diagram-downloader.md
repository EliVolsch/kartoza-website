## Overview

The SG Diagram Downloader is a QGIS plugin developed by Kartoza with sponsorship from Kirchhoff Surveyors. It enables users to download official cadastral diagrams from the South African Surveyor General's office directly within QGIS.

This tool is essential for surveyors, property developers, and GIS professionals working with South African land parcels.

## Key Features

- **Direct Download**: Download SG diagrams without leaving QGIS
- **Parcel Selection**: Select parcels from the map to retrieve diagrams
- **Batch Processing**: Download multiple diagrams in one operation
- **Search by Reference**: Find diagrams using SG reference numbers
- **PDF Integration**: View downloaded diagrams immediately

## Use Cases

- Land surveying projects
- Property development research
- Cadastral data verification
- Title deed investigations
- Municipal planning

## South Africa Specific

This plugin is designed specifically for the South African cadastral system and connects to the Surveyor General's online services. A valid connection to the SG services is required.

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "SG Diagram Downloader"
3. Click **Install Plugin**
4. Configure your connection settings
