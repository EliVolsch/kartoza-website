## Overview

trends.earth is a QGIS plugin developed with Conservation International that enables users to track changes in land productivity, land cover, and soil organic carbon. The tool supports the United Nations Sustainable Development Goal 15.3 (Life on Land) by providing standardised methods for monitoring land degradation neutrality.

## Key Features

- **Productivity Analysis**: Calculate vegetation productivity trends using satellite imagery
- **Land Cover Change**: Track changes in land cover over time using ESA CCI and other datasets
- **Soil Carbon**: Estimate changes in soil organic carbon stocks
- **SDG 15.3 Reporting**: Generate compliant reports for UNCCD and other international frameworks
- **Custom Area Analysis**: Analyse any region of interest with user-defined boundaries
- **Time Series Analysis**: Compare conditions across different time periods

## Use Cases

- National SDG reporting for UNCCD
- Conservation planning and monitoring
- Land restoration project evaluation
- Environmental impact assessment
- Climate change adaptation planning

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "trends.earth"
3. Click **Install Plugin**
4. The plugin will appear in your QGIS toolbar
