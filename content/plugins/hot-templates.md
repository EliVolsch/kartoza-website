## Overview

The HOT Templates and Symbology Manager is a Kartoza plugin developed for the Humanitarian OpenStreetMap Team (HOT). It provides standardised map templates, symbology, and print layouts for disaster response and humanitarian mapping operations.

## Key Features

- **Template Library**: Access pre-built map templates
- **Symbology Manager**: Standardised humanitarian symbols
- **Print Layouts**: Ready-to-use print compositions
- **Quick Setup**: Apply templates to existing projects
- **HOT Standards**: Compliance with HOT mapping standards
- **Localisation**: Multi-language support

## Template Categories

- Disaster response maps
- Situation awareness maps
- Infrastructure mapping
- Population distribution
- Accessibility maps
- Reference maps

## Use Cases

- Disaster response coordination
- Humanitarian needs assessment
- Field team briefing maps
- Situation reports
- Community mapping projects
- Missing Maps activations

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "HOT Templates"
3. Click **Install Plugin**
4. Access from the Plugins menu

## About HOT

The [Humanitarian OpenStreetMap Team](https://www.hotosm.org/) is an international team dedicated to humanitarian action and community development through open mapping.
