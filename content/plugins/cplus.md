## Overview

CPLUS (Climate Positive Land Use Strategy) is a decision support tool developed by Kartoza with Conservation International. The plugin helps users make informed decisions using spatial information such as land cover, carbon stocks, and potential for carbon sequestration to identify key areas for intervention and investment.

## Key Features

- **Carbon Stock Mapping**: Visualise current carbon stocks across landscapes
- **Sequestration Potential**: Identify areas with high carbon sequestration potential
- **Land Use Analysis**: Analyse land cover and land use patterns
- **Investment Prioritisation**: Rank areas for conservation and restoration investment
- **Scenario Modelling**: Compare different land use intervention scenarios
- **Report Generation**: Create detailed reports for stakeholders

## Use Cases

- Nature-based solutions planning
- Carbon credit project site selection
- Conservation prioritisation
- Land restoration planning
- Climate adaptation strategy development

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "CPLUS"
3. Click **Install Plugin**
4. Access the tool from your QGIS toolbar
