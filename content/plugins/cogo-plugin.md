## Overview

The CoGo (Coordinate Geometry) Plugin is a Kartoza tool developed for surveying clients, designed for land surveyors and cadastral professionals. It provides traditional surveying workflows within QGIS, allowing users to create parcels using bearings, distances, and coordinate input.

With over 32,000 downloads, this plugin is trusted by surveying professionals worldwide.

## Key Features

- **Bearing & Distance Entry**: Enter traverse data using bearings and distances
- **Coordinate Input**: Direct coordinate entry for precise positioning
- **Parcel Construction**: Build closed parcels from survey data
- **Closure Calculation**: Calculate traverse closures and adjustments
- **Area Computation**: Automatic area and perimeter calculation
- **Export Options**: Export to various cadastral formats

## Survey Methods Supported

- Traverse surveys
- Radiation surveys
- Intersection calculations
- Coordinate transformations
- Bearing-bearing intersections
- Bearing-distance intersections

## Use Cases

- Land surveying
- Cadastral mapping
- Property boundary definition
- Mining claim delineation
- Construction layout
- As-built surveys

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "CoGo"
3. Click **Install Plugin**
4. Access from the Vector menu or toolbar
