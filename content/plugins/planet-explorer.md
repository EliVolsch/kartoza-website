## Overview

Planet Explorer is a QGIS plugin developed by Planet Inc that provides seamless access to Planet's satellite imagery catalog. Kartoza provides ongoing maintenance and support for this plugin, ensuring compatibility with the latest QGIS versions and Planet API updates.

The plugin enables users to search, preview, and download high-resolution satellite imagery including PlanetScope, RapidEye, and SkySat data.

## Key Features

- **Imagery Discovery**: Search Planet's archive with spatial and temporal filters
- **Basemap Access**: Stream Planet Basemaps directly into QGIS
- **Order Management**: Place and track imagery orders
- **Preview Streaming**: Preview imagery before downloading
- **AOI Drawing**: Draw areas of interest for precise data selection
- **Composite Tool**: Combine multiple images into seamless mosaics
- **Publishing Stage Filters**: Access latest PlanetScope data filters

## Supported Products

- PlanetScope (3-4m resolution)
- RapidEye (5m resolution)
- SkySat (50cm resolution)
- Planet Basemaps (monthly and quarterly)

## Use Cases

- Agricultural monitoring
- Environmental change detection
- Urban development tracking
- Disaster response
- Conservation monitoring

## Requirements

- Active Planet account with API access
- QGIS 3.10 or later
- Python 3.6 or later

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "Planet Explorer"
3. Click **Install Plugin**
4. Enter your Planet API key to authenticate
