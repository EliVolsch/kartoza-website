## Overview

The Cadasta QGIS plugin enables users to work with the Cadasta platform for documenting land and resource rights. Developed by Kartoza for the Cadasta Foundation, this plugin provides a bridge between QGIS and the Cadasta cloud platform, supporting community-based land rights documentation efforts worldwide.

## Key Features

- **Project Management**: Create and manage Cadasta projects
- **Data Synchronisation**: Upload and download project data
- **Parcel Mapping**: Map land parcels and boundaries
- **Party Recording**: Document rights holders and relationships
- **Relationship Mapping**: Link parties to parcels
- **Offline Support**: Work offline and sync when connected

## Workflow Support

The plugin supports the complete land rights documentation workflow:

1. Create project structure
2. Map land parcels
3. Record community members
4. Document tenure relationships
5. Capture supporting evidence
6. Upload to Cadasta platform

## Use Cases

- Community land rights documentation
- Customary tenure mapping
- Informal settlement upgrading
- Land administration projects
- Rights-based conservation
- Participatory mapping

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "Cadasta"
3. Click **Install Plugin**
4. Configure your Cadasta account credentials

## About Cadasta

The [Cadasta Foundation](https://cadasta.org/) works to support communities around the world to document their land and resource rights.
