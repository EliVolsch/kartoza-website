## Overview

Species Explorer is an in-house Kartoza plugin that provides a simple interface for accessing the Global Biodiversity Information Facility (GBIF) database. Search for any species and instantly visualise occurrence records on your map.

The plugin is ideal for conservation practitioners, researchers, and biodiversity professionals who need quick access to species distribution data.

## Key Features

- **Species Search**: Find species by scientific or common name
- **Taxonomic Browser**: Navigate the taxonomic hierarchy
- **Occurrence Mapping**: Display occurrence points on your map
- **Data Filtering**: Filter by date, location, and data quality
- **Layer Creation**: Create QGIS layers from search results
- **GBIF Integration**: Direct connection to GBIF's REST API

## Data Access

The plugin connects to GBIF, providing access to:

- Over 2 billion occurrence records
- 400+ million species occurrence points
- Data from 2,000+ institutions worldwide
- Citizen science observations
- Museum specimen records

## Use Cases

- Conservation planning
- Environmental impact assessment
- Species distribution modelling
- Biodiversity research
- Educational projects
- Invasive species monitoring

## Installation

1. Open QGIS and go to **Plugins → Manage and Install Plugins**
2. Search for "Species Explorer"
3. Click **Install Plugin**
4. Access from the QGIS menu or toolbar
